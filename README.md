## System Purpose
This project implements a Knowledge-Based System (Expert System) for Loan Approval. 
It simulates a bank loan officer's decision-making using logical rules in SWI-Prolog.
The system takes user inputs such as income, credit score, and employment status,and provides a decision: Approved, Conditionally Approved, or Rejected.
